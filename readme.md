#Installation:
```
$ opam install dune  
$ make  
$ make exec  
```

Controls:
* w -> Up
* s -> Down
* a -> Left
* d -> Right
* ; -> Select
* space -> Step
* c -> Cancel
* b -> Construct building
* q -> Quit
