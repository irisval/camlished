- If dune isn't installed on your system then run "$ opam install dune"
- "make" to build
- "make exec" to launch game
- "make test" to run tests
- In order to generate documentation, odoc must be installed. If not installed
  on your system, run "$ opam install odoc" before "make docs" to generate docs
- Docs will be found at _build/default/_doc
